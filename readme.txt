**HTML WP Menu addon is a plugin that create wp menu from html.**


== Description ==
HTML WP Menu addon is a plugin that create wp menu from html. these plugin can only fetch single menu...submenus will not fetched.

Plugin Link: [HTML WP](https://wordpress.org/plugins/html-wp/) 
Website Link: [HTML WP](https://html-wp.com) 



= Main Features: =

* Automatically Menu creation from html .

* Assign menus link with page.





== Installation ==
First, install the plugin. After activation, you will see a added option on html menu screen.



== Frequently Asked Questions ==
= How html menu will be automatically created? =

Html menu on file will be created automatically with synching of menu item.
